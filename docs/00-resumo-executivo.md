# Resumo executivo (Sprint 1)
Totem com IA embarcada para shoppings que:
1) Entrega **rota e promoções** em até ~2s (voz ou toque).
2) Gera **QR/link** para continuar no celular, sem app.
3) Permite **lojas publicarem promoções** em minutos, tornando o projeto rentável.

Valor: experiência rápida/personalizada para visitantes + canal de mídia mensurável para lojas/shopping.
